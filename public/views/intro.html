Intro

Congratulations, you're close running your own pagerbot on heroku!

Before we're done however, we need to set up a few things:
  • Connect to pagerduty.
  • Set up how pagerbot connects to your channels.
  • Select which commands (plugins) pagerbot should answer to.
  • Set up short aliases for coworkers and schedules to refer them by.

Navigating through the following pages and setting everything up should take around 8 minutes.
